# 主流 B 端组件库参考

## 组件库总览

| 组件库 | 出品方 | 技术栈 | 适用场景 | 文档首页 |
|------|------|------|---------|---------|
| **Ant Design** | 阿里蚂蚁 | React | 企业级中后台，生态最完善，设计规范成熟 | https://ant-design.antgroup.com/components/overview-cn |
| **Arco Design** | 字节跳动 | Vue / React | 现代感强，组件丰富，适合内部工具和SaaS | https://arco.design/vue/docs/start |
| **Element Plus** | 饿了么 | Vue 3 | Vue生态主流，上手成本低，社区庞大 | https://element.eleme.cn/#/zh-CN/component/installation |
| **TDesign** | 腾讯 | Vue Next / React | 企业级规范完善，多端一致性好 | https://tdesign.tencent.com/vue-next/overview |
| **Semi Design** | 字节抖音 | React | 设计系统完备，暗色模式支持好 | https://semi.design/zh-CN |
| **Naive UI** | 社区 | Vue 3 | 轻量灵活，TypeScript友好 | https://www.naiveui.com/zh-CN/os-theme |
