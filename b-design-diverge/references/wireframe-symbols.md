# ASCII/Unicode 示意图规范

## 符号速查表

| 元素类型 | 符号 |
|--------|------|
| 单选（选中） | `◉` |
| 单选（未选） | `○` |
| 复选（选中） | `☑` |
| 复选（部分） | `⊡` |
| 复选（未选） | `□` |
| 输入框 | `[____________]` |
| 下拉选择 | `[选项 ▾]` |
| 主按钮 | `[■ 确认]` |
| 次按钮 | `[ 取消 ]` |
| 标签/Tag | `[标签 ×]` |
| 开关（开） | `◉——` |
| 开关（关） | `——○` |
| 图标占位 | `⊙` `⊕` `⊗` |
| 分割线 | `─────────────` |
| 面包屑 | `首页 > 模块 > 当前页` |
| 表格行 | `│ 内容 │ 内容 │ 操作 │` |
| 折叠（收） | `▶ 标题` |
| 折叠（展） | `▼ 标题` |
| 树节点 | `└─` `├─` |
| 步骤条 | `① → ② → ③` |
| 进度条 | `[████████░░] 80%` |
| 徽标数字 | `(3)` |

## 常用布局模板

### 筛选器 - 分组折叠式
```
┌─────────────────────────────────┐
│ 筛选条件                    [收起]│
├─────────────────────────────────┤
│ ▼ 状态                          │
│    ◉ 全部   ○ 启用   ○ 禁用     │
│ ▶ 时间范围                      │
│ ▶ 所属部门                      │
├─────────────────────────────────┤
│ [■ 查询]  [ 重置 ]              │
└─────────────────────────────────┘
```

### 列表页 - 工具栏 + 表格
```
┌─────────────────────────────────────────┐
│ [■ 新建]  [批量导入]        [🔍 搜索___]│
├────┬──────┬──────┬──────┬───────────────┤
│ □  │ 名称 │ 状态 │ 时间 │    操作       │
├────┼──────┼──────┼──────┼───────────────┤
│ ☑  │ 示例A│ 启用 │ 今天 │ 编辑 停用 删除│
│ □  │ 示例B│ 禁用 │ 昨天 │ 编辑 启用 删除│
└────┴──────┴──────┴──────┴───────────────┘
  已选1项  [批量启用] [批量删除]      共20条
```

### 表单 - 双列布局
```
┌─────────────────────────────────────────┐
│ 基本信息                                │
├────────────────┬────────────────────────┤
│ 姓名 *         │ 所属部门 *             │
│ [__________]   │ [请选择 ▾]            │
│ 手机号         │ 角色                   │
│ [__________]   │ [请选择 ▾]            │
├─────────────────────────────────────────┤
│              [■ 保存]  [ 取消 ]         │
└─────────────────────────────────────────┘
```

### 穿梭框 - Transfer
```
┌──────────────┐        ┌──────────────┐
│ 待选 (12)    │        │ 已选 (3)     │
├──────────────┤  [>]   ├──────────────┤
│ □ 产品部     │  [<]   │ ☑ 研发部     │
│ □ 市场部     │        │ ☑ 运营部     │
│ □ 财务部     │        │ ☑ 法务部     │
└──────────────┘        └──────────────┘
```

### 步骤表单
```
① 基本信息 ──→ ② 权限配置 ──→ ③ 确认提交

┌─────────────────────────────────┐
│ 步骤2：权限配置                  │
│                                  │
│ 功能权限                         │
│   ☑ 查看   ☑ 编辑   □ 删除      │
│   □ 审批   □ 导出               │
│                                  │
│ [ 上一步 ]            [■ 下一步] │
└─────────────────────────────────┘
```

### 抽屉 - Drawer
```
主页面内容（变暗）         ┌──────────────────┐
                           │ 编辑成员          │
                           ├──────────────────┤
│                          │ 姓名 *            │
│                          │ [______________]  │
│                          │                   │
│                          │ 角色              │
│                          │ [请选择 ▾]       │
│                          ├──────────────────┤
│                          │[■ 保存]  [ 取消 ] │
                           └──────────────────┘
```
