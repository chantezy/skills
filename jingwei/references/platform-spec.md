# 精卫平台（4399云文档 & 4399项目管理）深度功能分析报告

> 分析时间：2026-06-04  
> 平台版本：v2.6.6  
> 分析方式：通过Agent控制浏览器实际操作、DOM解析、页面截图

## 一、产品概述

| 项目 | 内容 |
| :--- | :--- |
| **产品名称** | 精卫（Jingwei） |
| **产品类型** | 企业级文档管理 + 项目协作一体化平台 |
| **所属行业** | 互联网/游戏行业内部工具（4399集团） |
| **核心用户** | 产品经理、交互设计师、开发团队、运营团队、客服团队 |
| **核心价值主张** | 将文档管理与项目管理深度融合，提供统一的知识沉淀和任务协作工作空间 |
| **解决的问题** | 团队信息分散、知识管理困难、项目进度不透明、跨团队协作效率低 |
| **差异化优势** | 文档与项目数据互通；支持多种文档类型（富文本、Markdown、甘特图）；企微消息通知集成 |

## 三、核心业务流程分析

### 3.1 文档管理核心流程

| 阶段       | 用户行为                  | 页面     | 系统动作                           |
| :------- | :-------------------- | :----- | :----------------------------- |
| **进入**   | 用户访问 `#/docDashboard` | 文档仪表盘  | 加载最近查看列表，显示文档库侧边栏              |
| **浏览**   | 切换标签页"最近查看"/"我的收藏"    | 文档仪表盘  | 刷新列表数据，切换显示列（查看时间 ↔ 收藏时间）      |
| **搜索**   | 点击"搜索文档"，输入关键词        | 全局搜索面板 | 展开高级搜索面板，支持按关键词/创建者/项目/版本/年份筛选 |
| **定位文档** | 在左侧文档库点击某个库           | 文档仪表盘  | 切换到该文档库的文档列表                   |
| **查看文档** | 点击文档标题                | 文档编辑器  | 打开文档详情页，加载ProseMirror编辑器       |
| **编辑文档** | 在编辑器中修改内容             | 文档编辑器  | 实时保存，支持富文本/Markdown/甘特图编辑      |
| **收藏**   | 点击星标图标                | 文档列表   | 将文档加入"我的收藏"，记录收藏时间             |
| **分享**   | 使用分享功能                | 文档编辑器  | 生成分享链接，支持文档库项目分享               |
| **协作**   | 邀请成员/成员管理             | 文档库设置  | 添加协作者，设置权限                     |

### 3.2 项目管理核心流程

| 阶段        | 用户行为                      | 页面     | 系统动作                 |
| :-------- | :------------------------ | :----- | :------------------- |
| **进入**    | 用户访问 `#/projectDashboard` | 项目仪表盘  | 加载我的任务、团队统计、项目列表     |
| **查看任务**  | 点击"查看更多"                  | 我的任务页面 | 展示完整任务列表，支持按执行者筛选    |
| **管理任务**  | 修改表头配置、筛选条件               | 我的任务页面 | 自定义列显示、保存配置          |
| **进入项目**  | 点击项目卡片                    | 项目详情页  | 加载项目看板/表格/甘特图视图      |
| **创建任务**  | 在项目中新建任务                  | 项目详情页  | 填写任务名称、优先级、截止时间、所属项目 |
| **跟踪进度**  | 切换看板/表格/甘特图视图             | 项目详情页  | 多维度展示任务进度            |
| **任务协作**  | 评论、上传附件                   | 任务详情页  | 支持文字评论、视频上传、文件附件     |
| **归档/回收** | 归档已完成任务                   | 项目详情页  | 移入归档区或回收站            |

## 五、用户角色分析

| 角色 | 目标 | 核心操作 | 权限范围 |
| :--- | :--- | :--- | :--- |
| **普通成员** | 查看文档、了解项目进度 | 搜索/查看文档、查看任务列表 | 对文档库有只读权限，对所属项目有查看权限 |
| **文档编辑者** | 创建和维护文档 | 创建/编辑/删除文档、管理文件夹 | 对指定文档库有读写权限 |
| **项目成员** | 参与项目任务执行 | 查看/更新任务状态、评论、上传附件 | 对所属项目有任务操作权限 |
| **项目管理员** | 管理项目和团队 | 创建项目、管理成员、配置字段、归档任务 | 对项目有完整管理权限 |
| **文档库管理员** | 管理文档库 | 创建文档库、邀请成员、设置权限 | 对文档库有完整管理权限 |
| **团队管理员** | 管理团队 | 创建团队、管理团队成员 | 团队级别的管理权限 |
| **超级管理员** | 平台全局管理 | 用户管理、权限分配、系统配置 | 平台所有功能的最高权限 |

## 六、技术架构分析

### 6.1 前端技术栈

| 技术 | 用途 |
| :--- | :--- |
| **Vue.js 2.x** | 前端框架（通过 `__vue__` 实例确认） |

| **SVG图标系统** | 自定义JwIcon组件 |

### 6.2 URL路由结构

| 路由 | 页面 | 说明 |
| :--- | :--- | :--- |
| `#/docDashboard` | 文档仪表盘 | 文档中心首页 |
| `#/projectDashboard` | 项目仪表盘 | 项目管理首页 |
| `#/myTask` | 我的任务 | 个人任务列表 |
| `#/project/:id` | 项目详情 | 具体项目看板/表格/甘特图 |
| `#/doc/:id` | 文档详情 | 文档编辑器页面 |

## 十、总结与评价

### 优势
1. **文档与项目深度融合**：文档库与项目管理共享同一平台，信息流转顺畅
2. **多视图支持**：项目管理支持看板、表格、甘特图三种视图，满足不同场景需求
3. **强大的搜索能力**：全局搜索支持5个维度的筛选条件
4. **丰富的文档类型**：支持富文本、Markdown、甘特图等多种文档格式
5. **企业级协作**：集成企微消息通知、浏览器推送等企业级功能

### 待改进
1. **项目详情页加载**：部分项目详情页存在加载缓慢或无法加载的问题
2. **文档点击交互**：文档列表中的点击事件响应有待优化（需通过Vue Router而非直接URL跳转）
3. **信息密度**：仪表盘信息密度较高，新用户可能需要一定的学习成本
