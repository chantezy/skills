# PRD 需求文档生成 Skill

产品需求文档（PRD）生成技能。

## 功能

帮助产品经理创建清晰、可操作的需求文档，包括：

- 结构化的需求收集工作流程
- PRD 最佳实践指导
- 完整的需求文档模板
- 中英文双语支持

## 使用场景

当用户请求以下内容时触发：

- "创建 PRD"
- "写产品需求文档"
- "帮我写一个功能需求"
- "Create a PRD for..."
- 任何与产品需求相关的请求

## 使用方法

直接请求生成 PRD：

```
帮我创建一个用户认证功能的 PRD
```

或

```
Create a PRD for dark mode feature
```

## 目录结构

```
skills-prd-generator/
├── SKILL.md                    # 技能主文件
├── references/
│   └── prd-template.md       # PRD 模板
└── README.md                  # 本文件
```

## 工作流程

1. **收集背景信息** - 通过发现式对话获取需求
2. **生成结构** - 按照标准PRD结构组织内容
3. **输出文档** - 生成 Markdown 格式的完整 PRD
