# 社群日常运营 SOP

## 适用对象
品牌福利群 / 客户服务群

## 周节奏
| 时间 | 动作 | 内容 | 目标 |
|---|---|---|---|
| 周一 9:30 | 早安播报 | 一周福利预告 | 暖群 |
| 周二 11:00 | 干货分享 | 产品使用技巧 / 行业知识 | 提供价值 |
| 周三 15:00 | 互动话题 | 提问 + 抽奖 | 拉互动 |
| 周四 20:00 | 直播预告 / 新品 | 内容种草 | 转化预热 |
| 周五 19:00 | 周末限时优惠 | 优惠券 + 商品链接 | 转化 |
| 周六 10:00 | 用户故事 | 老客户晒单 / 好评 | 社交证明 |
| 周日 16:00 | 答疑专场 | 群主集中回答 | 信任维护 |

## 日常规则
- 群规置顶
- 广告 / 飞单 → 警告 + 踢出
- 新人入群 → 自动欢迎 + @ 群主

## 角色分工
- 群主：维护节奏 + 答疑
- 助手：素材发布 + 数据统计
- KOC：氛围带动（埋点）

## 关键指标
- 日活跃率（发言 UV / 群人数）
- 群转化 GMV
- 退群率
