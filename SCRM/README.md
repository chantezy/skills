# 私域 SCRM 方案发散 Skill

> 一套为「游戏社区平台」量身打造的 AI 知识工程，
> 让 AI 真正读懂你的业务语言，输出可落地的私域 SCRM 方案。

---

## 🎯 这是什么

这不是一份普通的方案文档，而是一个 **结构化的 AI Skill 知识库**。

把它喂给 AI（ChatGPT / Claude / 自建大模型），AI 就能：

- 听懂私域 SCRM 的专业黑话（活码、SOP、RFM、会话存档…）
- 站在 **游戏社区平台** 视角，而不是普通零售视角思考
- 主动对标网易云商 / 微盛 / 微伴 等竞品
- 输出 **业务流程图 + 功能清单 + 数据埋点 + 企微接口 + 风险点** 的完整方案
- 给出"产品 / 技术 / 运营"三方下一步行动清单

---

## 💡 核心优势

### 1. 工程化而非文档化
不是一份大长文，而是 36+ 个分工明确的 .md 文件，AI 按需检索调用，
输出更准、更稳、更专业。

### 2. 概念-功能双向映射
专业术语词典不只给"定义"，每个词条都附带：
**定义 + 功能要素 + 企微接口 + 常见坑 + 竞品差异**，
AI 一查就能直接输出可落地的功能流程。

### 3. 游戏社区平台深度定制
不套用零售 / 教育模板，独立设计了：
**玩家 / 创作者 / 公会 / 开发者** 四类用户的差异化运营体系，
覆盖内容生态、社区氛围、KOL/KOC、商业化变现等平台独有玩法。

### 4. 竞品视角内置
内置 5 家主流 SCRM 竞品深度档案，AI 每次方案都会主动标注
"行业通用做法 + 我们的差异化机会点"。

### 5. 标准工作流约束输出质量
4 套标准工作流（功能发散 / 竞品对标 / 问题诊断 / 技术评估）+
正反例对照，让 AI 输出节奏统一、不糊弄、不空谈。

### 6. 合规优先
内置企微平台规则 + 数据隐私 + 游戏行业特殊合规清单，
凡涉及加好友 / 群发 / 会话存档的方案，AI 主动校验风险。

## 🚀 快速上手

### 三步开始用

**第一步：把 SKILL.md 内容贴到对话开头**
告诉 AI："以下是你要遵循的工作 Skill，请按其中的规则回答我后续的问题。"

**第二步：按需补充上下文**
如果讨论具体功能，可以再把相关 `references/` 或 `workflows/` 文件粘贴进来。

**第三步：用结构化方式提问**
参考下面"提问模板"。

### 推荐提问模板

```
[上下文] 我们是游戏社区平台（参考 game_community/overview.md）。
[任务类型] 功能发散 / 竞品对标 / 问题诊断 / 技术评估
[具体问题] 我想设计一个「创作者孵化」功能，目标是 3 个月把月发文创作者数提升 50%。
[期望输出] 按 workflows/feature_brainstorm.md 的标准流程输出。
```

## 🎮 业务定位提醒

我们是 **游戏社区平台**（类似 TapTap / 小黑盒 / B站游戏区），
**不是** 游戏开发商。

所以以下话题 **不适用**：
- ❌ 单款游戏的拉新付费链路
- ❌ 游戏内 IAP（内购）转化优化
- ❌ 渠道买量 ROI

我们真正关心：
- ✅ 多游戏用户的聚合沉淀
- ✅ 内容生态（攻略/视频/直播/评测）
- ✅ 创作者经济
- ✅ 公会与社区氛围
- ✅ 平台商业化（广告 / CPS / 电商 / 内容付费）
