# 模块：客户运营 (Operation)

## 定位
**运营动作执行层**，所有触达客户的动作都在这里。

## 子功能详解

### 1. 群发消息
- 群发朋友圈：批量发圈，可按标签分组可见
- 群发私信：1V1 私信批量发送
- 群发模板：常用群发模板复用

**约束**：受企微频控（朋友圈 1/天，私信每客户 1/天）

### 2. 营销活动
- 活动列表：所有活动汇总
- 活动创建：拖拽 / 表单式创建
- 活动数据：报名/参与/转化数据

**典型活动类型**：抽奖、礼包码发放、签到、问卷、报名

### 3. 客户旅程（重点）
- 旅程设计：可视化拖拽设计自动化流程
- 触发条件：标签变化 / 行为触发 / 时间触发
- 自动化流程：T+N 时序触达

**典型旅程**：
- 新人 7 日 SOP
- 流失召回 SOP
- 首单后 SOP
- 创作者孵化 SOP

### 4. 内容中心
- 素材库：图文 / 视频 / 海报 / 小程序
- 话术库：分类管理 + 一键发送
- 海报模板：批量生成专属海报

## 与其他模块的关系

```
客户管理（提供客户+标签）→ 客户运营（执行动作）→ 数据分析（评估效果）
内容中心 → 提供素材 → 群发/活动/旅程
```

## 适合发散的扩展方向

### 玩家运营
- 礼包码发放工具（与开发商合作）
- 新游内测招募
- 游戏赛事报名

### 创作者运营
- 创作者任务派发
- 流量扶持工具
- 商业化订单撮合

### 公会运营
- 公会活动报名
- 公会会长群批量沟通
- 公会扶持计划

### AI 能力增强
- AI 智能话术推荐（侧边栏）
- AI 朋友圈文案生成
- AI 旅程优化建议（基于历史数据推荐节点）

### 旅程能力增强
- A/B 测试
- 旅程分支条件细化
- 跨渠道触达（企微 + APP push + 短信）

## 给 AI 的提示
- 这里管的是"做什么"
- 单次触达 → 群发；多次时序触达 → 客户旅程；有特定主题 → 营销活动
- 不要把"运营动作"塞到客户管理模块
