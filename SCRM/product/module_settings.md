# 模块：系统设置 (Settings)

## 定位
**平台治理层**，企业级配置和管理。

## 子功能

### 1. 企业设置
- 企业信息
- 品牌配置（Logo / 主题色）
- 接口配置（第三方对接）

### 2. 权限管理
- 角色管理（自定义角色）
- 权限分配（功能权限）
- 数据权限（部门 / 个人 / 全部）

### 3. 日志管理
- 操作日志
- 登录日志
- 系统日志

### 4. 其他设置
- 通知设置
- 集成配置
- 备份恢复

## 适合发散的扩展方向
- 多企业 / 多品牌矩阵管理（针对未来横向扩展）
- 审批流配置（敏感操作审批）
- API 开放平台（给开发商接入）

## 给 AI 的提示
- 这里是**管理员域**，使用频率低但重要
- 任何"配置、权限、日志、品牌"类需求落这
