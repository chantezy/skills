# 技术架构速览

## 前端
- 基于企业微信 JS-SDK
- 响应式设计（PC + 移动）
- 模块化组件开发

## 后端
- RESTful API 设计
- 企业微信 API 集成
- 消息队列处理（异步任务）

## 数据
- 客户数据主从同步
- 行为数据实时采集
- 数据分片存储

## 集成点
- 企业微信开放平台
- 4399 游戏盒 APP（核心数据源）
- 4399 统一登录
- 第三方（广告 / 电商 / 短信）

## 给 AI 的提示
- 方案设计要考虑"前端 vs 后端 vs 数据"分工
- 涉及大批量触达要走消息队列
- 实时性要求高 → API 推送；非实时 → 离线计算
