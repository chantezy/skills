# 模块：数据分析 (Analytics)

## 定位
**数据驱动决策层**，沉淀所有数据并产生洞察。

## 子功能详解

### 1. 客户数据
- 客户增长（趋势 / 渠道来源）
- 客户分布（地区 / 性别 / 兴趣 / 活跃度）
- 客户价值（RFM / LTV）

### 2. 运营数据
- 消息触达（送达率 / 已读率）
- 活动转化（参与率 / 转化率）
- 互动数据（点击 / 评论 / 分享）

### 3. 销售数据
- 销售业绩（GMV / 订单数）
- 转化漏斗（曝光 → 点击 → 加购 → 下单）
- 订单分析（客单价 / 复购）

### 4. 数据报表
- 日报 / 周报 / 月报（自动生成）
- 自定义报表（拖拽配置）

## 适合发散的扩展方向

### 游戏行业专属
- 游戏兴趣分布
- 玩家价值分层（白嫖 / 小 R / 大 R）
- 内容消费偏好（攻略 / 视频 / 直播）
- 创作者数据中心（粉丝增长 / 内容效果 / 收益）
- 公会数据看板

### 高级分析
- 客户旅程漏斗分析（旅程节点流失率）
- AB 测试结果对比
- 归因分析（多渠道来源归因）
- 流失预警模型

### AI 洞察
- AI 自动数据日报（异常指标自动告警 + 归因分析）
- AI 客户分群推荐
- AI 内容效果预测

## 给 AI 的提示
- 这里是**消费者**，不要把分析逻辑塞到业务模块
- 涉及"看数据 / 出报表 / 找原因"都在这
- 数据指标定义参考 `references/metrics_dictionary.md`
