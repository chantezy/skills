# SCRM 指标口径词典

## 一、获客类
| 指标 | 公式 | 口径说明 |
|---|---|---|
| 扫码量 | 扫码 PV | 含重复扫码 |
| 添加申请量 | 发起添加好友的人数 | UV |
| 添加通过率 | 通过数 / 申请数 | 反映话术与头像质量 |
| 加粉成本 CAC | 渠道花费 / 新增通过数 | 含人力 + 物料 + 广告 |
| 渠道 ROI | 渠道带来 GMV / 渠道花费 | T+30 计算 |

## 二、运营类
| 指标 | 公式 | 备注 |
|---|---|---|
| 触达率 | 已读数 / 发送数 | 朋友圈用"曝光" |
| 互动率 | 互动数（点赞+评论+点击）/ 触达 | |
| 群活跃率 | 当日发言 UV / 群人数 | |
| SOP 完成率 | 实际执行 / 应执行 | 反映员工执行力 |

## 三、转化类
| 指标 | 公式 | 备注 |
|---|---|---|
| 首单转化率 | 首单数 / 新好友数 | 通常 T+30 |
| 复购率 | 复购客户数 / 总下单客户数 | 时间窗口需指定 |
| 客单价 | GMV / 订单数 | |
| LTV | 客单价 × 复购次数 × 毛利率 | 12 个月 LTV 最常用 |

## 四、客户资产类
| 指标 | 公式 | 备注 |
|---|---|---|
| 客户留存率 | 仍是好友数 / 历史添加总数 | 反推流失 |
| 流失率 | 删除 + 拉黑 / 总好友数 | |
| 标签覆盖率 | 至少 1 个业务标签的客户数 / 总数 | 衡量客户档案完整度 |

## 五、AI 与质检
| 指标 | 公式 | 备注 |
|---|---|---|
| 质检覆盖率 | 已质检会话 / 总会话 | AI 质检通常达 100% |
| 飞单识别准确率 | TP / (TP+FP) | |
| 话术推荐采纳率 | 采纳数 / 推荐数 | 反映 AI 实用性 |
