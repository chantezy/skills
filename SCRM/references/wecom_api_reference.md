# 企业微信开放接口能力清单

> 用途：当 AI 设计方案时，需要校验功能是否可由企微官方接口实现。
> 注意：本文档为概念级映射，具体参数以企微开放文档为准（dev.work.weixin.qq.com）。

---

## 一、客户联系（外部联系人）

### 添加联系我（活码）
- 接口：`externalcontact/add_contact_way`
- 实现功能：渠道活码、群活码、单人活码
- 限制：单企业总数有上限、单员工挂码数有限
- 坑：员工离职需调用 `transfer_customer` 转移

### 获取客户列表 / 详情
- 接口：`externalcontact/get_external_contact`
- 实现功能：客户档案、客户标签查询
- 关键字段：external_userid、unionid、来源、添加时间、备注

### 客户标签
- 接口：`externalcontact/mark_tag` / `get_corp_tag_list`
- 实现功能：自动打标、手动打标、标签管理
- 限制：企业标签 vs 个人标签分开

### 群发 / SOP
- 接口：`externalcontact/add_msg_template`
- 实现功能：客户 SOP、群发消息
- 限制：员工需在企微 App 二次确认；每客户每天 1 次

### 朋友圈
- 接口：`externalcontact/add_moment_task`
- 实现功能：企业朋友圈批量发布
- 限制：员工每天 1 条企业圈

### 客户群
- 接口：`externalcontact/groupchat/*`
- 实现功能：群活码、群运营、群成员管理
- 限制：单客服最多 2000 群

### 离职继承
- 接口：`externalcontact/resigned/transfer_customer`
- 实现功能：员工离职客户继承
- 限制：客户有 3 天反悔期

---

## 二、会话存档

### 会话存档（合规录制）
- 接口：`msgaudit/*`
- 实现功能：会话存档、智能质检、飞单检测
- 强制要求：员工授权 + 客户端显示提示
- 价格：按账号收费

---

## 三、应用与侧边栏

### 自建应用
- 能力：可在企微工作台 + 聊天侧边栏中嵌入 H5
- 实现功能：导购工作台、订单查询、商品推荐、话术库
- 鉴权：OAuth2 + JS-SDK 获取 external_userid

### 通讯录
- 接口：`user/*` / `department/*`
- 实现功能：组织架构同步、员工管理

## 四、能力 - 功能映射表

| 想做的功能 | 能否原生支持 | 关键接口 |
|---|---|---|
| 渠道活码 | ✅ | add_contact_way |
| 群活码 | ✅ | groupchat + contact_way |
| 自动欢迎语 | ✅ | add_contact_way 配置 |
| 自动打标 | ✅ | mark_tag |
| 客户 SOP | ✅ | add_msg_template + 业务调度 |
| 朋友圈批量发布 | ✅ | add_moment_task |
| 会话存档 | ✅（付费） | msgaudit |
| 离职继承 | ✅ | transfer_customer |
| 批量加好友 | ⚠️（需引导客户主动添加） | 不能强加，只能扫码 |
| 智能话术 | ⚠️（需自建 AI） | 接口提供数据，AI 自建 |
| 商品 / 订单同步 | ⚠️（业务系统对接） | 不在企微范畴 |

## 五、限频提示（高频踩坑）

| 接口 | 频控 |
|---|---|
| 群发 | 每客户每员工每天 1 次 |
| 朋友圈 | 每员工每天 1 条 |
| 活码挂码数 | 单员工上限有限 |
| 客户群数 | 单客服 2000 群 |

---

## 六、给 AI 的使用提示

- 设计任何功能前，先在本文件中查询是否原生支持。
- 若"需自建"，需在方案中明确"业务系统侧的承接逻辑"。
- 涉及群发 / 朋友圈时主动提示频控限制。
