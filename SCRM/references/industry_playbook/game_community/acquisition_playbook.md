# 获客玩法

## 一、外部流量入口矩阵

| 渠道 | 内容形式 | 引流路径 | 适合人群 |
|---|---|---|---|
| 抖音/B站游戏区 | 视频/直播 | 主页活码 → 加企微 | 兴趣玩家 |
| 小红书 | 攻略图文 | 评论引流 → 私信 → 企微 | 二次元/休闲玩家 |
| 微博游戏话题 | 热点讨论 | 活动 H5 → 企微 | 跟热点玩家 |
| 微信公众号 | 深度内容 | 文末活码 | 硬核玩家 |
| SEO/百度 | 攻略/评测 | 落地页 → 企微 | 工具型用户 |
| KOL 合作 | 推荐位 | 专属海报 | KOL 粉丝 |
| 线下漫展/电竞赛事 | 摊位/扫码 | 活码 + 周边 | 核心玩家 |
| 游戏厂商联动 | 礼包码 | 礼包页 → 加企微领取 | 该游戏玩家 |

## 二、活码分流策略（重点）

### 分流维度（这是和零售很不一样的地方）

```
活码分流维度
├─ 来源渠道（抖音 / 小红书 / 公众号 ...）
├─ 游戏兴趣（从落地页提交的兴趣标签）
├─ 设备 OS（iOS / Android，影响内购引导）
├─ 用户角色（玩家 / 创作者 / 公会会长）
└─ 地域（影响线下活动推荐）
```

### 典型活码设计
- **泛玩家活码**：所有玩家通用，进新人群
- **游戏垂类活码**：每款游戏一个，进对应游戏群
- **创作者专属活码**：从创作者中心页面跳转
- **公会入驻活码**：B 端通道
- **赛事活码**：每场赛事一个，便于复盘 ROI

## 三、内容引流玩法（核心）

### 1. 攻略 SEO + 文末活码
- 高 SEO 价值内容（"XX 游戏新手攻略"）
- 文末"加群获取完整攻略 + 大佬解答"

### 2. 礼包码引流
- 与开发商合作首发独家礼包
- 用户加企微领取，自动打"XX 游戏玩家"标签

### 3. KOL 探店式引流
- 邀请 KOL 在视频中露出"加群有专属福利"
- 给每个 KOL 唯一活码，便于结算

### 4. 创作者扶持反向引流
- 创作者发文带平台活动话题
- 引导粉丝加官方企微

### 5. 赛事 / 联动引流
- 与游戏开发商联合举办赛事
- 报名必须加企微，进赛事专属群

## 四、裂变玩法（游戏行业特化）

### 1. 礼包码裂变
- 邀请 3 人加企微，解锁稀有礼包码
- 重点：礼包码必须是真稀缺（与开发商深度合作）

### 2. 公会招募裂变
- 公会会长拉人进公会群
- 平台给会长流量/福利奖励

### 3. 内测资格裂变
- 邀请好友 → 解锁新游内测资格
- 这是游戏行业最有效的诱因

### 4. 装备/道具抽奖裂变
- 与开发商合作的稀有道具
- 邀请越多抽奖次数越多

## 五、关键指标

| 指标 | 口径 | 健康参考值 |
|---|---|---|
| 加粉成本 | 渠道花费 / 新增企微好友 | 游戏行业 5-30 元 |
| 通过率 | 通过数 / 申请数 | >70% |
| 7 日留存 | 新增第 7 天仍活跃 | >40% |
| 内容触达率 | 触达数 / 总粉丝数 | >60% |
| 渠道 ROI（T+30） | 渠道 GMV / 渠道成本 | >1.5 |

## 六、给 AI 的提示
- 游戏行业获客的核心诱饵是 **稀缺资源**（礼包码、内测资格、独家内容、稀有道具）
- 不要套用零售的"优惠券"思路，玩家对优惠券不敏感
- 兴趣分流非常重要，泛粉群活跃度差异巨大
- KOL/创作者反向引流是低成本获客主力
