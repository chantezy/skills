# 内容生态运营

## 一、内容生态全景

```
内容供给（创作者 + 官方 + UGC）
        ↓
内容分发（推荐 + 群 + 朋友圈 + 私信）
        ↓
内容消费（玩家阅读 / 互动）
        ↓
内容反馈（数据 → 反哺创作者）
```

## 二、内容类型矩阵

| 类型 | 生产成本 | 消费场景 | 商业化价值 |
|---|---|---|---|
| 新游评测 | 高 | 决策期 | 高（CPS） |
| 游戏攻略 | 中 | 游戏中 | 中 |
| 短视频/集锦 | 中 | 碎片化 | 高（广告） |
| 直播 | 高 | 黄金时段 | 高（打赏+广告） |
| 资讯/快讯 | 低 | 日常 | 低 |
| 玩家故事/段子 | 低 | 社交 | 低 |
| 深度专题 | 极高 | 硬核玩家 | 高（品牌） |

## 三、私域中的内容分发

### 1. 兴趣群定向分发
- 群标签 + 内容标签匹配
- 每天 1-2 条精品内容推送

### 2. 朋友圈精选
- 每日 1 条平台精选
- 按兴趣分组可见（避免打扰）

### 3. 1V1 个性化推送（高级玩法）
- 基于用户行为打标
- 重要内容触发推送
- 频次控制：每周不超过 2 次

### 4. 创作者新内容通知
- 关注关系 + 创作者发文 = 自动私信
- 必须用户主动开启

## 四、UGC 激励机制

### 1. 数据回报
- 浏览/点赞/评论数据实时反馈
- 周报：本周创作者数据

### 2. 流量倾斜
- 新人创作者首发流量包
- 优质内容算法加权

### 3. 物质激励
- 创作激励金（按播放/阅读）
- 任务奖励（命题创作）
- 商业化分成

### 4. 荣誉体系
- 创作者等级
- 月度榜单
- 年度盛典

## 五、内容供给的私域抓手

### 攻略征集
- 群内发起："本周征集 XX 游戏新手攻略，入选送游戏激活码"

### 评测招募
- 新游内测期招募评测员
- 私域是最快的招募渠道

### UGC 命题
- 每周话题挑战赛
- 群内/朋友圈话题预告

### KOL 群（重点）
- 把头部创作者沉淀进 KOL 群
- 重大热点第一时间同步
- 内部选题会、爆料、独家素材

## 六、内容质量控制

### 审核机制
- 一级机审：违规词、低质内容
- 二级人审：抽样 + 举报触发
- 三级精选：编辑推荐位

### 反作弊
- 刷量识别
- 抄袭检测
- 多账号矩阵识别

## 七、给 AI 的提示
- 内容生态是飞轮，不是某个功能
- 私域 SCRM 在内容生态里的角色是 **"创作者运营 + 精准分发 + UGC 召集"**
- 不要把朋友圈做成广告位，会毁掉信任
