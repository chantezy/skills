# 平台商业化变现

## 一、收入模型矩阵

| 收入来源 | 类型 | 占比建议 | 私域中的玩法 |
|---|---|---|---|
| 广告 | B 端 | 30-50% | 信息流广告、品牌专区 |
| 游戏分发 CPS | B 端 | 20-30% | 礼包码 / 渠道追踪 |
| 电商（周边/道具/点卡） | C 端 | 10-20% | 商城 + 私域种草 |
| 内容付费 | C 端 | 5-10% | 付费攻略 / 课程 / 会员 |
| 直播打赏 | C 端 | 5-15% | 抽成 |
| 创作者商业撮合 | B+C | 5-10% | 平台撮合 + 抽佣 |
| 赛事 / 活动 | B 端 | 5-10% | 商务合作 |

## 二、广告变现（B 端为主）

### 广告形式
- 信息流广告（游戏推荐位）
- 品牌专区（新游首发）
- 开屏 / 横幅
- 创作者植入（撮合）

### 私域中的广告变现
- 朋友圈广告（克制使用！）
- 群内品牌活动（联动福利发放）
- 1V1 精准推送（高定向）

### 关键：用户体验平衡
- 广告频次控制
- 内容广告比例不超过 1:10
- 高价值用户广告降频

## 三、游戏 CPS 分发

### 模式
- 平台导流 → 用户下载游戏 → 充值 → 平台按比例分成

### 私域链路
1. 评测内容种草
2. 私域礼包码发放（与开发商合作）
3. 引导下载（带渠道号）
4. 充值后台归因
5. 月度结算

### 关键
- 选品：与平台用户匹配的游戏
- 内容：优质评测带动转化
- 渠道追踪：UTM 完整闭环

## 四、电商变现（C 端）

### 商品类型
- 游戏周边（手办 / 服饰 / 鼠标垫）
- 游戏点卡 / 道具
- 游戏内充值
- 二手交易（账号 / 装备）

### 私域玩法
- 商城 + 企微导购侧边栏
- 群内秒杀 / 团购
- 朋友圈种草
- 私信精准推荐

### 关键
- 周边类客单价高、复购低
- 点卡类客单价低、高频
- 不同品类不同运营节奏

## 五、内容付费（C 端）

### 形式
- 付费攻略 / 评测（深度内容）
- 课程（吃鸡技巧 / MOBA 进阶）
- 会员订阅（去广告 + 独家内容 + 福利）
- 数字内容包（壁纸 / 原画集）

### 私域玩法
- 免费内容引流 → 付费内容转化
- 会员专属群
- 优惠券触达

## 六、直播变现

### 形式
- 打赏抽成
- 直播带货（联动电商）
- 直播广告

### 私域玩法
- 直播预告私信通知
- 直播间专属福利
- 主播粉丝群

## 七、创作者商业撮合

### 模式
- 品牌方需求 → 平台匹配创作者 → 创作者执行 → 平台抽佣

### 私域中的角色
- KOL 经纪人 1V1
- 品牌方商务群
- 订单 / 结算工具

## 八、付费用户 SOP

### 首次付费引导
- 新用户 T+7：低门槛付费引导（9.9 元会员体验）
- 高互动用户：精准推荐符合兴趣的付费内容

### 复购运营
- 付费用户单独打标 + 高优先级运营
- 专属客服群
- 续费提醒 SOP

### VIP 体系
- 累计付费金额分级
- 等级特权（专属客服 / 内测 / 周边）

## 九、关键指标

| 指标 | 含义 |
|---|---|
| ARPU | 月度人均收入 |
| ARPPU | 付费用户人均收入 |
| 付费率 | 付费 UV / 总 UV |
| CPS 转化率 | 私域用户中下载并充值的比例 |
| 广告 eCPM | 千次曝光收入 |
| LTV | 用户生命周期价值 |

## 十、给 AI 的提示
- 不要简单把零售卖货思路套到游戏平台
- B 端（广告+CPS）通常是收入大头
- C 端付费要轻量级切入（9.9/19.9 会员）
- 任何商业化都要平衡用户体验，否则破坏内容生态
