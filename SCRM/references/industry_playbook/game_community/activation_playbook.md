# 激活与留存玩法

## 一、新用户 7 日激活 SOP（玩家版）

| 时间 | 动作 | 内容 | 目标 |
|---|---|---|---|
| T+0（5分钟内） | 1V1 自动私信 | 欢迎语 + 新人礼包码 + 兴趣选择问卷 | 建立印象 + 收集兴趣 |
| T+0 后 30 分钟 | 自动打标 + 拉群 | 根据兴趣进对应游戏垂类群 | 兴趣匹配 |
| T+1（10:00） | 朋友圈 | 平台精选内容（图文+视频） | 内容种草 |
| T+2（20:00） | 1V1 私信 | 个性化攻略 / 热门视频推荐 | 内容消费 |
| T+3 | 群消息 | 新人欢迎 + 群规 + 大佬带新 | 社交融入 |
| T+5 | 1V1 私信 | 邀请参与平台任务（签到送礼包） | 行为激活 |
| T+7 | 数据评估 | 已激活 → 进入常态运营；未激活 → 流失干预 SOP | 分层 |

## 二、激活的核心动作（按效果排序）

1. **兴趣匹配**：进了对的群比加 10 个群有用
2. **首次内容消费**：第一次刷到喜欢的攻略 / 视频
3. **首次互动**：评论、点赞、回复
4. **首次 UGC**：发第一条内容、提第一个问题
5. **首次社交关系**：关注创作者 / 加入公会
6. **首次商业化**：领礼包、买周边、买内容

## 三、留存玩法

### 1. 日活打卡体系
- 签到 → 积分 → 兑换礼包码 / 游戏道具
- 关键：积分必须能换到玩家真想要的东西

### 2. 任务体系（轻游戏化）
- 新手任务：完善资料、关注创作者、加入群
- 日常任务：签到、评论、分享
- 进阶任务：发布 UGC、参与活动
- 周任务：连续活跃奖励

### 3. 兴趣群常态运营
- 见 `community_operation.md`

### 4. 创作者关注机制
- 关注的创作者发新内容 → 主动私信通知
- 把"创作者"做成留存抓手

### 5. 公会归属感
- 加入公会的用户留存远高于无公会用户
- 公会工具包提升公会粘性

### 6. 赛事/活动节奏
- 周末小活动、月度大活动、季度赛事
- 每个活动都是召回老用户的机会

## 四、流失干预

### 流失预警模型（建议字段）
- 最近 7 天活跃天数 < 1
- 最近 14 天 UGC 数 = 0
- 关注的创作者更新没看
- 群发消息 30 天未读

### 召回 SOP
| 阶段 | 动作 | 内容 |
|---|---|---|
| 沉睡 7-14 天 | 私信 | 关心 + 推他偏好的内容 |
| 沉睡 14-30 天 | 朋友圈 | 平台大事件 / 新游首发 |
| 沉睡 30-60 天 | 私信 | 稀缺礼包码召回 |
| 沉睡 60 天+ | 降频 + 大型节点触达 | 春节 / 周年庆 |

## 五、关键指标

| 指标 | 健康参考值 |
|---|---|
| 次日留存 | >50% |
| 7 日留存 | >40% |
| 30 日留存 | >25% |
| 群活跃率 | >15% |
| 月人均 UGC | >0.5 条 |

## 六、给 AI 的提示
- 激活 ≠ 加好友，激活是"产生第一次有意义的互动"
- 兴趣匹配是激活的命门，匹配错了一切白搭
- 留存靠"内容 + 社交 + 任务"三轮驱动，缺一轮就掉
