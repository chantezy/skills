# 社区运营与群运营

## 一、群矩阵设计

### 群类型
| 群名 | 定位 | 规模上限 | 运营节奏 |
|---|---|---|---|
| 新人欢迎群 | 7 日内新用户暂留 | 200 | 高频引导，7 日后转分流 |
| 游戏垂类群（按 IP） | 单款游戏深度讨论 | 500（多群矩阵） | 日常话题 + 资讯 |
| 兴趣品类群（MOBA/FPS/二次元） | 跨游戏兴趣聚合 | 500 | 周话题 + 推荐 |
| 核心玩家群 | 平台 VIP / 精英 | 200（邀请制） | 内测特权、官方反馈 |
| KOL/创作者群 | 创作者沟通 | 500 | 选题、热点、扶持 |
| 公会会长群 | B 端运营对接 | 200 | 公告、活动招募 |
| 赛事/活动专属群 | 活动期限定 | 临时 | 活动结束 7 日内解散或转化 |

## 二、群运营 SOP（以游戏垂类群为例）

| 时段 | 动作 | 内容示例 |
|---|---|---|
| 早 9:00 | 早安播报 | "XX 游戏今日新闻 / 维护信息" |
| 中午 12:00 | 攻略分享 | 1 条精选攻略 |
| 晚 19:00 | 热点话题 | "今天的版本更新你怎么看？" |
| 晚 21:00 | 直播/视频推荐 | KOL 直播预告 |
| 周五晚 | 福利日 | 礼包码抽奖 |
| 周日晚 | 一周复盘 | 本周热点 + 下周预告 |

## 三、群氛围建设

### 1. 群文化塑造
- 群规清晰（禁广告 / 禁飞单 / 禁外群引流）
- 群昵称规范（地区+ID+主玩游戏）
- 入群仪式感（欢迎语 + @ 群主）

### 2. 群角色分工
| 角色 | 职责 |
|---|---|
| 群主（官方运营） | 节奏把控、官方信息、答疑 |
| 副群主（核心玩家） | 氛围带动、新人答疑 |
| 神秘嘉宾（KOL/开发者） | 不定期空降 |
| 群机器人 | 入群欢迎、关键词回复、签到 |

### 3. 互动设计
- 每日话题（机器人定时发）
- 周末活动（抽奖、竞猜、晒图）
- 节点活动（新版本、赛事、节日）

## 四、公会运营（游戏行业特色）

### 公会价值
- 自带 50-500 名成员，组织力强
- 是赛事 / 活动承接的关键节点
- 公会会长往往是 KOC

### 平台赋能
1. **公会工具包**（小程序）
   - 成员管理、签到、活动报名、内部排行榜
2. **公会扶持计划**
   - 优质公会流量倾斜
   - 赛事 / 活动优先邀请
3. **公会会长群**
   - 平台与会长直接沟通
   - 活动招募 / 资源对接

### 公会分级
| 等级 | 标准 | 权益 |
|---|---|---|
| 普通 | <50 人 | 基础工具 |
| 优质 | 50-200 人 + 周活跃 60% | 流量推荐 |
| 金牌 | 200+ 人 + 月活跃 70% + UGC | 赛事承接、独家资源 |
| 钻石 | 头部公会 | 战略合作、年度签约 |

## 五、关键指标

| 指标 | 健康值 |
|---|---|
| 群日活跃率（发言 UV / 群人数） | >15% |
| 新人 7 日留群率 | >70% |
| 群转化率（群内→平台 DAU） | >60% |
| 公会数 / 公会人数 | 持续增长 |

## 六、给 AI 的提示
- 不要建一堆死群，宁少而精
- 群主能力是关键，机器人替代不了人
- 公会是游戏社区独有的强组织节点，必须重视
