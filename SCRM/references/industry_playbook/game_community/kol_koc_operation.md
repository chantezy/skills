# KOL / KOC 体系

## 一、定义与差异

| 类型 | 定义 | 在平台的作用 | 运营策略 |
|---|---|---|---|
| KOL（意见领袖） | 公域影响力大、专业内容输出 | 拉新引流、品牌背书 | 1V1 经纪人 + 签约 |
| KOC（关键意见消费者） | 群内 / 圈内有影响力、真实玩家 | 社区氛围、口碑传播、UGC | 群运营 + 荣誉激励 |

## 二、KOL 体系（详见 `creator_operation.md`）

主要是创作者中的腰部 + 头部。

## 三、KOC 识别（重点）

### KOC 识别信号
- 在群里高频发言、被高频 @
- UGC 内容自然获赞高
- 老玩家、注册时长长
- 公会会长 / 副会长
- 朋友圈被转发多

### 自动识别规则（建议落地）
```
规则：满足以下 3 项中至少 2 项 → 自动打标"KOC 苗子"
1. 群内 30 日发言 >50 条 且 被 @ >5 次
2. 月 UGC ≥3 条 且 平均互动率 >5%
3. 注册 ≥180 天 且 月活 ≥20 天
```

## 四、KOC 运营

### 1. 荣誉激励
- 平台勋章 / 称号
- 月度 KOC 榜单
- 年度盛典邀请

### 2. 特权激励
- 新游内测优先
- 平台决策内测群（产品改版征求意见）
- 内部资讯优先获取

### 3. 物质激励（轻量）
- 周边礼品
- 游戏道具 / 点卡
- 不强调现金（避免变味）

### 4. 社交激励（最有效）
- 与官方运营熟络
- 与 KOL 同框
- 线下活动邀请

## 五、KOC 在私域的核心价值

```
对内：维护群氛围、答疑、带新
对外：自发传播、UGC、口碑
对上：用户声音的真实反馈
```

## 六、KOC → KOL 转化

部分 KOC 有内容能力，可以转化为创作者：
1. 私信邀请尝试发内容
2. 提供选题 + 流量扶持
3. 数据反馈 + 鼓励
4. 进入创作者孵化通道

## 七、避免的坑

- ❌ KOC 给现金奖励 → 变味，吸引投机者
- ❌ 把 KOC 当工具人使唤 → 失去真诚
- ❌ KOL/KOC 混淆 → 运营策略错位
- ❌ 不维护 KOC 关系 → 流失到竞品
