# 网易云商 SCRM

## 一句话定位
网易系老牌通信和客服服务商转型 SCRM，强在"全渠道触达 + 客户档案 + 客服整合"。

## 核心优势
- 通信能力强：短信、外呼、邮件、企微一站式
- 客服整合：呼叫中心 + 在线客服 + 企微客服
- 全渠道客户档案
- 网易品牌背书，企业客户信任度高

## 核心功能
- 渠道活码、群活码
- 客户档案（多渠道聚合）
- 客户 SOP / 群发
- 智能客服机器人
- 短信 / 外呼 / 邮件多触点
- 会话存档与质检
- 数据看板

## 适合企业
- 中大型企业
- 有呼叫中心 / 客服中心
- 重视"多通道触达"的业务（金融、零售、教育）

## 我们可以借鉴 / 差异化
- 借鉴：通信通道整合思路
- 差异化：更聚焦企微生态深度、AI 智能化更前沿
